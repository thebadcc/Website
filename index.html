BAD Token v0.1


The BAD token is an Ethereum-based token built using the ERC20 standard. This token was created to facilitate participation and coordination in the development of the THEBADCC, and eventually the handoff of community governance.  The BAD token is a multi-utility token required for application to THEBADCC, an incentivized voting game to dictate access to a suite of supplementary services exclusive to network members. It’s also used to signal support or disapproval for proposals made by community participants. The inherent value to any given person will differ depending on the extent of their desire to participate in the community.

The BAD Token is not premined and relys on a merge mining function to create its supply. Merged mining is the process of mining multiple different cryptocurrencies at the same time. In our case, the block reward is split into three categories for incentivizing the community, including the block miner, the message sender, and the owner of the contract. The BAD Token does not have a hard cap, rather the community will regulate supply by adjusting the block rewards through governance.

Token holders will be able to vote on proposals created by THEBADCC using Snapshot. Eventually, we anticipate migrating governance to the Aragon v2 protocol.
